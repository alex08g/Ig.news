## Blog sobre o mundo da tecnologia🌏💻

<img src="https://github.com/alex08g/Ig.news/blob/main/public/ig.news-example.png" alt="Ig.news" width="100%" height="100%"/>

Projeto desenvolvido durante o ignite da Rocketseat🚀💜.

### Tecnologias utilizadas

* [Nextjs](https://nextjs.org/)
* [Sass](https://sass-lang.com/)
* [Prismic](https://prismic.io/)
* [Stripe](https://stripe.com/en-br)


### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```
* github auth configurado
* stripe configurado
* prismic configurado

### Instalação

1. Faça o clone do repositório
   ```sh
   git clone https://github.com/alex08g/Ig.news
   ```
2. Instale os pacotes
   ```sh
   npm install
   ```
3. crie e configure o arquivo `.env` baseado no `.env.example`
